# LifeGood 記帳理財

> 打造完美人生，從記帳與理財開始。

## 簡述

LifeGood 是專為台灣使用者打造的 iOS 生活管理 App，整合記帳、理財、人生三大模式。記帳涵蓋收支與週期投射；理財管理儲蓄險、股票、載具與房地產並附閃卡稀有度；人生記錄履歷里程碑、家庭成員與房地產地圖。三端資料雙向連動，資料只存本機與 iCloud 不上傳第三方伺服器，支援 JSON/CSV 匯出匯入，並可串接財政部電子發票 API 自動匯入消費紀錄，需 iOS 17 以上。

## 功能概述

- 記帳模式
    - 總覽：本月收支摘要、今日花費、分類統計、最近交易
    - 收入：5 大分類（薪水/獎金/禮金/確幸/投資），固定薪水自動代入每月
    - 變動支出：10 大分類，預設顯示近一週紀錄，可逐週展開
    - 固定支出：8 大分類、月/季/年週期投射，房貸車貸自動試算利率
    - 圖表：日/週/月/季/年五維度趨勢
    - 設定：模式切換、匯率設定、匯出 JSON/CSV、匯入、一小時前資料復原、清除

- 理財模式
    - 總覽：總資產、四大類別卡片、配置比例、每月現金流
    - 儲蓄險：TWD/USD 幣別、複利年利率、期滿領回試算
    - 股票：成本/市值/損益/報酬率，買賣連動銀行或證券帳戶
    - 汽車：動力類型連動、閃卡稀有度、定期/變動支出分開管理
    - 房地產：購入/估值以萬元輸入，多筆貸款、已支出、變動支出、淨現金流；人生分頁記錄房屋資料/水電瓦斯/保險/附屬資產
    - 圖表：資產配置圓餅、股票損益、房產績效

- 人生模式
    - 總覽：個人檔案閃卡（傳說級，固定置頂）、最近里程碑
    - 履歷：依分類分節顯示，配偶章節置頂；家庭成員自動衍生結婚/離婚/出生
    - 家庭：配偶/爸媽/兒女/兄弟姐妹/其他親屬管理
    - 財富：銀行（含信用卡縮排）、證券、保險管理，帳戶餘額與圖表
    - 房地產：台灣地圖顯示大頭針、點擊動畫展開物件列表、縣市分組
    - 設定：與其他模式共用

- 跨模式連動
    - 理財資產雙向同步對應的固定/變動支出
    - 記帳項目可反向連結既有理財資產
    - 收入/支出可指定扣款目標（銀行/信用卡），自動連動帳戶圖表
    - 人生家庭成員連動個人檔案與里程碑
    - 人生房地產連結理財物件並以地圖呈現

- 自動化整合
    - 電子發票自動匯入：連結財政部手機條碼載具，自動匯入消費紀錄並依商家分類至變動支出
    - 內建 60+ 條商家分類規則，支援使用者自訂
    - 可選擇整張發票合併或拆分成各品項；可一鍵撤銷已匯入的發票
