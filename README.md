# LifeGood 記帳理財

> 打造完美人生，從記帳與理財開始。

## 簡述

LifeGood 是專為台灣使用者打造的 iOS 生活管理 App：記帳、理財、人生三合一。長按麥克風一句話 AI 記帳，自動辨識金額/分類/同行人；變動支出支援語音、搜尋與地點自動完成。理財含股票 T+2 交割、配股/配息聯動收入；房地產卡片整合裝潢照片、PDF/Excel 文件與水電收據集錦，並分「理財／房屋資料／資產」三分頁，資產可依樓層管理物件。全 App 台幣金額過萬以「萬」、過千萬以「千萬」顯示。名片支援 Apple Vision OCR、多電話/Email/傳真、一鍵加入聯絡人。資料只存本機與 iCloud，需 iOS 17 以上。

## 功能概述

- 記帳模式
    - 總覽：本月收支摘要、今日花費、分類統計、最近交易
    - 收入：5 大分類（薪水/獎金/禮金/確幸/投資），固定薪水自動代入每月
    - 變動支出：10 大分類，預設顯示近一週紀錄，可逐週展開
    - 固定支出：8 大分類、月/季/年週期投射，房貸車貸自動試算利率
    - 圖表：日/週/月/季/年五維度趨勢
    - 設定：模式切換、匯率設定、匯出 JSON/CSV、匯入、一小時前資料復原、清除

- 理財模式
    - 總覽：總資產、四大類別卡片、配置比例、每月現金流
    - 儲蓄險：TWD/USD 幣別、複利年利率、期滿領回試算
    - 股票：成本/市值/損益/報酬率，買賣連動銀行或證券帳戶
    - 汽車：動力類型連動、閃卡稀有度、定期/變動支出分開管理
    - 房地產：卡片分「理財／房屋資料／資產」三分頁。理財含購入/估值、多筆貸款、已支出、變動支出、淨現金流；房屋資料含水電瓦斯/保險/附屬資產與照片、PDF/Excel 文件集錦；資產分頁可依樓層新增、管理物件
    - 圖表：資產配置圓餅、股票損益、房產績效

- 人生模式
    - 總覽：個人檔案閃卡（傳說級，固定置頂）、最近里程碑
    - 履歷：依分類分節顯示，配偶章節置頂；家庭成員自動衍生結婚/離婚/出生
    - 家庭：配偶/爸媽/兒女/兄弟姐妹/其他親屬管理
    - 財富：銀行（含信用卡縮排）、證券、保險管理，帳戶餘額與圖表
    - 職涯：部屬管理（自動評分、請假紀錄）、公司組織圖；部屬班表棋盤式燈號排班，支援大夜 / 小夜輪班範本與班別時間自訂
    - 房地產：台灣地圖顯示大頭針、點擊動畫展開物件列表、縣市分組
    - 設定：與其他模式共用

- 跨模式連動
    - 理財資產雙向同步對應的固定/變動支出
    - 記帳項目可反向連結既有理財資產
    - 收入/支出可指定扣款目標（銀行/信用卡），自動連動帳戶圖表
    - 人生家庭成員連動個人檔案與里程碑
    - 人生房地產連結理財物件並以地圖呈現

- 自動化整合
    - 電子發票自動匯入：連結財政部手機條碼載具，自動匯入消費紀錄並依商家分類至變動支出
    - 內建 60+ 條商家分類規則，支援使用者自訂
    - 可選擇整張發票合併或拆分成各品項；可一鍵撤銷已匯入的發票

## 版本紀錄

主版本號迭代重點，口語版。

- v18（2026/06）

    - 班別系統：大夜班 / 小夜班 / 假日值班 / 時差假 / 休息，班別時間（平日 / 假日）可自訂
    - 一鍵套用班表範本：大夜班輪班（時差假→6 天大夜→休息，共 8 天）、小夜班（週一至週五 5 天，自動跳過週末）；單日可直接清除班別
