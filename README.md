# LifeGood 記帳理財

> 打造完美人生，從記帳與理財開始。

LifeGood 是一款專為台灣使用者設計的 iOS 記帳理財 App，整合日常支出管理與資產配置追蹤。透過設定頁面一鍵切換「記帳模式」與「理財模式」，兩端資料智慧連動。

## 雙模式功能

### 記帳模式

| Tab | 功能 |
|-----|------|
| 總覽 | 本月收支摘要（收入/支出/餘額）、三欄摘要卡片、今日花費、分類統計、最近交易 |
| 收入 | 4 大分類（薪水/獎金/禮金/確幸），薪水支援固定/非固定切換和自動標題（M604格式），週期設定，月收入試算 |
| 變動支出 | 10 大分類（飲食/交通/汽車/娛樂/購物/日用品/醫療/教育/社交/其他），篩選、分組、新增/編輯/刪除 |
| 固定支出 | 8 大分類，週期設定（月/季/年），智慧投射到每個月。保險分 5 種子分類，貸款分 3 種子分類，所有類別可選擇關聯理財資產 |
| 圖表 | 日/週/月/季/年五種趨勢，長條+折線混合圖，支出比例分析 |
| 設定 | 模式切換、匯出 JSON/CSV、匯入、資料統計、清除、版本資訊 |

### 理財模式

| Tab | 功能 |
|-----|------|
| 總覽 | 總資產、四大類別卡片（儲蓄險/股票/汽車/房地產）、資產配置比例、每月現金流 |
| 儲蓄險 | 保單管理，支援 TWD/USD 幣別、複利年利率、自動計算期滿領回與目前價值 |
| 股票 | 持股管理，成本/市值/損益/報酬率 |
| 汽車 | 車輛管理，動力類型（油車/電車/混合動力）決定變動支出選項，定期支出（車貸/稅費/訂閱）與變動支出分開管理 |
| 房地產 | 物件管理，購入價/估值以萬元輸入，多筆貸款（各有期數）、已支出房屋金額、變動支出，月租金、淨現金流、租金報酬率 |
| 圖表 | 資產配置圓餅圖、股票損益柱狀圖、房產績效、儲蓄險摘要 |
| 設定 | 與記帳模式共用 |

## 跨模式連動

| 記帳操作 | 理財自動同步 |
|---------|------------|
| 固定支出 > 保險 > 儲蓄險 | 建立理財儲蓄險紀錄 |
| 固定支出 > 貸款 > 房貸 | 建立理財房地產紀錄 |

| 理財操作 | 記帳自動同步 |
|---------|------------|
| 新增儲蓄險 | 建立固定支出（保險-儲蓄險） |
| 房地產填入房貸 | 建立固定支出（貸款-房貸） |
| 汽車新增定期支出 | 建立固定支出（車貸/稅費/訂閱） |
| 汽車新增變動支出 | 建立變動支出（汽車-油錢/停車/保養等） |

| 記帳操作 | 理財自動同步 |
|---------|------------|
| 變動支出關聯「汽車」 | 新增至對應汽車的變動支出 |
| 變動支出關聯「股票」 | 建立股票持股紀錄 |
| 變動支出關聯「儲蓄險」 | 連結至已有的儲蓄險保單 |
| 變動支出關聯「房地產」 | 新增至對應房地產的變動支出 |
| 固定支出選「保險-儲蓄險」 | 建立理財儲蓄險紀錄 |
| 固定支出選「貸款-房貸」 | 建立理財房地產紀錄 |
| 固定支出選「貸款-車貸」 | 新增至對應汽車的定期支出 |
| 固定支出關聯「汽車」 | 新增至對應汽車的定期支出 |
| 固定支出關聯「房地產」 | 連結至已有的房地產物件 |
| 固定支出關聯「儲蓄險」 | 連結至已有的儲蓄險保單 |

所有連動皆為雙向，新增、編輯、刪除任一邊都會同步更新對方。刪除時連結的項目也會一併移除。

## 專案結構

```

│   │       ├── SavingsInsuranceView.swift  # 儲蓄險列表
│   │       ├── AddSavingsInsuranceView.swift
│   │       ├── StockView.swift            # 股票列表
│   │       ├── AddStockView.swift
│   │       ├── VehicleView.swift          # 汽車列表

│   │       ├── RealEstateView.swift       # 房地產列表

│   ├── Assets.xcassets/
│   └── Preview Content/
├── AppStoreConnect.md
└── README.md
```

## 安裝與執行

1. Clone 此專案
2. 使用 Xcode 15+ 開啟 `LifeGood.xcodeproj`
3. 選擇 iOS Simulator 或實機
4. Build & Run (Cmd + R)

無需安裝任何第三方套件，所有功能皆使用 Apple 原生框架。
